## love2d 版运行说明

基于游泳的鱼的金庸群侠传 lua 复刻版，utf8 编码，纯 lua(love2d) 版.

### 运行

```
cd bin

# if using AppImage
# love.AppImage .
```

```bash
cd src && lua tests/test_runner.lua
```
